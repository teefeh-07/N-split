# WalletConnect Integration

## Overview

SereneHub integrates WalletConnect for secure wallet connections across multiple platforms.

## Setup

1. Install dependencies:
```bash
npm install @reown/appkit @reown/appkit-adapter-wagmi
```

2. Configure the provider in your app.

## Supported Wallets

- Leather (Hiro Wallet)
- Xverse
- OKX Wallet
- And any WalletConnect-compatible wallet

## Troubleshooting

### Connection Issues

1. Clear browser cache
2. Try a different browser
3. Ensure wallet app is updated

### QR Code Not Scanning

1. Ensure good lighting
2. Refresh the page and try again
