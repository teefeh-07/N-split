# Deployment Guide

## Prerequisites

- Node.js 18+
- Clarinet CLI
- GitHub account

## Environment Setup

```bash
cp .env.example .env
# Edit .env with your configuration
```

## Contract Deployment

1. Navigate to contracts directory
2. Run `clarinet check`
3. Deploy to testnet

## Frontend Deployment

```bash
cd frontend
npm run build
npm run start
```
